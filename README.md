## 📜 Proje Hakkında

**Focus Hero**, Flutter ile geliştirilmiş, son derece interaktif ve oyunlaştırılmış bir Pomodoro sayacıdır. Monoton ders çalışma veya iş seanslarını destansı RPG macerasına dönüştürür. Sadece süre tutmak yerine:

- ⚔️ **Canavarlarla savaş**
- 📈 **Seviye atla ve XP kazan**
- 🐺 **Pet yumurtaları çatla**
- 💰 **Altın toplayarak marketten eşya satın al**
- 🔥 **Günlük serinizi koruyun**
- 👥 **Arkadaşlarınızla yarışın (liderlik tablosu)**

Proje, **Clean Architecture** (Temiz Mimari) ve **BLoC State Management** kullanılarak kurgulanmış, **production-ready** bir uygulamadır.

## 🎮 Oyun Mekanikleri

### Savaş Sistemi
1. **Focus Session Başlat** → 5-120 dakika seç
2. **Odaklan** → Her saniye boss'a hasar ver
3. **Uygulama Tabı Değiştir** → CEZA! (15 HP kaybı)
4. **Oturum Bitir** → Ödüller Kazanavsız!

### XP & Leveling
```
Level 1-10 : 100 XP per level
Level 11-20: 150 XP per level
Level 21+  : 200 XP per level
```

### Pet Evolution
```
🥚 Yumurta (600 saniye)
  ↓
🐺 Wolf / 🦇 Bat / 🐉 Dragon
  ↓
Seviye 1-10 (Her 500 XP'de +1 Level)
```

## 📜 About The Project

**Focus Hero** is a highly interactive, gamified Pomodoro timer built with Flutter. It transforms monotonous study or work sessions into epic RPG adventures. Instead of just tracking time, you:

- ⚔️ **Battle monsters**
- 📈 **Level up and earn XP**
- 🐺 **Hatch pet companions**
- 💰 **Collect gold for the shop**
- 🔥 **Maintain your daily streak**
- 👥 **Compete with friends on the leaderboard**

The project is built using **Clean Architecture** and **BLoC State Management**, making it a **production-ready** application.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| ⚔️ **Boss Battles** | Every minute of focus = damage dealt. 10+ unique boss types. |
| 📈 **RPG Mechanics** | Level system, XP, Gold, experience points. |
| 🐺 **Pet System** | Buy eggs → Hatch → Level up → Gain power in battle. |
| 🛒 **Item Shop** | Purchase XP Shield, Freeze Spell, Health Potion, 4+ themes. |
| 🎵 **Ambient Sounds** | Rain, Tavern, Fire, Forest, Ocean, Space... (10+ options) |
| 🔥 **Streak Tracking** | Daily focus records, completed quests, 30-day calendar. |
| 🌍 **Social Features** | Global leaderboard, friend system, comparisons. |
| 📱 **Ad System** | AdMob integration (Banner, Interstitial, Rewarded ads). |
| 🎮 **Gamification** | Achievements, Milestones, Achievement badges. |

---

## 🎮 Game Mechanics

### Battle System
1. **Start Focus Session** → Choose 5-120 minutes
2. **Focus** → Deal damage every second
3. **Switch App Tab** → PENALTY! (15 HP loss)
4. **Complete Session** → Earn Rewards!

### XP & Leveling
```
Level 1-10 : 100 XP per level
Level 11-20: 150 XP per level
Level 21+  : 200 XP per level
```

### Pet Evolution
```
🥚 Egg (600 seconds)
  ↓
🐺 Wolf / 🦇 Bat / 🐉 Dragon
  ↓
Level 1-10 (Gain +1 Level every 500 XP)
```
